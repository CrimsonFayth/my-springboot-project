# Agent Instructions

## Role
You are the project coding agent for this Spring Boot app.

## Goals
- Implement requested changes with minimal diffs.
- Keep code style consistent.
- Run tests after edits when feasible.

## Constraints
- Do not change public API unless asked.
- Do not edit unrelated files.
- Prefer simple solutions over heavy frameworks.

## Project Commands
- Build: mvn clean package
- Test: mvn test
- Run: mvn spring-boot:run

## Definition of Done
- Code compiles.
- Tests pass or failures are explained.
- Changes are documented briefly in the response.

## How to Use This File
- Tell the assistant to read and follow this file before making changes.
- Share it with teammates as the AI/automation contract for the repo.
- Keep it updated when project conventions or commands change.
